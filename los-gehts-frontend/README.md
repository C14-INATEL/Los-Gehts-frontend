# 🚀 Los Gehts Frontend

Aplicação frontend desenvolvida com **Next.js + React**.

---

## 🧰 Tecnologias

- ⚛️ React
- ▲ Next.js
- 🟦 TypeScript
- 🎨 Tailwind CSS
- ✅ ESLint

---

## 📦 Instalação

Clone o repositório:

```bash
git clone https://github.com/seu-usuario/los-gehts-frontend.git
cd los-gehts-frontend
npm install
npm run dev

```
src/
 ├── app/          # Rotas (App Router)
 ├── components/   # Componentes reutilizáveis
 ├── services/     # Integrações com APIs
 ├── hooks/        # Hooks customizados
 ├── utils/        # Funções auxiliares
 └── styles/       # Estilos globais
